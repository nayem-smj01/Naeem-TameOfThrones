# Tame of Thrones

This App takes input the secret message and
outputs the name of kingdoms for valid case

## Requirements
1. Java 11+
2. Gradle (to compile)

## Testes on OS
1. Ubuntu 
1. Windows

## How to run
``
java -jar geektrust.jar <path_to_input_file>
``

Example
``
eg:  java  -jar  geektrust.jar resources/input1.txt
``
